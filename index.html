FAQ

よくあるご質問

初期費用はかかりますか？ +
いいえ。契約成立まで一切の費用は発生しません。初回のご相談・課題整理も完全無償です。参考として、RPOは月額30万円〜、CHRO代行は月額50万円〜、HRTech導入支援は月額20万円〜が目安です。ご予算に合わせた柔軟なプランをご提案します。
どのような企業規模に対応していますか？ +
50名規模のスタートアップから500名以上の中堅企業まで幅広く対応しています。業界も問いません。
契約期間の縛りはありますか？ +
最低契約期間は原則3ヶ月ですが、プロジェクトの内容に応じて柔軟に設定可能です。
RPOとCHRO代行の違いは何ですか？ +
RPOは採用プロセスの設計・実行を代行するサービスです。CHRO代行は採用に限らず、人事制度設計・組織設計・評価制度など、人事戦略全体を経営視点で設計・推進するサービスです。
自社にHRTechツールがすでにある場合でも相談できますか？ +
はい。既存ツールの活用改善や、追加ツールの選定・連携設計もご支援しています。
リモートでの対応は可能ですか？ +
はい。オンラインでの打ち合わせ・支援を基本としており、全国どこからでもご利用いただけます。
メンバーはどのような経歴の方ですか？ +
PASONA・PERSOL・RECRUIT・JACなど大手人材会社出身のプロフェッショナル100名で構成されています。
相談から開始までどのくらいかかりますか？ +
最短1週間で支援を開始できます。まずはお気軽にご相談ください。
料金はどのように決まりますか？ +
採用課題の内容と支援範囲に応じて、最適なプランを個別にお見積もりします。料金表の金額はあくまで目安です。まずは無料相談で現状の課題をお聞かせください。その場で概算の目安をお伝えできます。
最低契約期間はありますか？ +
プランや支援内容によって異なります。短期のスポット支援から中長期の伴走型まで柔軟に設計できますので、無料相談時に貴社の状況に合わせてご案内します。
途中で解約することはできますか？ +
可能です。解約条件や予告期間はプランにより異なるため、契約前に書面で明確にご提示します。「合わなければやめられるか不安」という方も、安心してご相談ください。
